Social Network

A gamers' social network built with Python. The program takes as input a string of sentences containing user-names, social connections and gaming preferences, and sets up a network in the form of a Python dictionary. It can retrieve and manipulate network data to provide useful information, such as:
  • Finding a path from one player in the network to another.
  • Listing the preferred games or connections of any given user.
  • Finding secondary connections (friends of friends).
  • Suggesting new social connections based on users' gaming preferences.

Marathon Map

Marathon Map is an interactive web application that showcases local attractions and amenities in Marathon, Florida -- the heart of the Florida Keys!

The project relies on the Google Maps API and Places Library for its basic functionality and content. The Knockout.js and jQuery libraries provide additional structure and streamlining.
